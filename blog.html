Question-1: What are the different ways to select an element in the DOM?

There are few ways to select an element in the dom such as getElementById(),getElementsByClassName(), getElementsByTagName(), querySelector(), querySElectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML gets the html markup and the text content, innerText gets only the content inside a html element, textContent also gets the content, it gets all the content regardless it is shown on the page or not.

Question-3: What is event delegation in the DOM?

Event delegation is the process of event bubbling to handle in the DOM.

Question-4: What is event bubbling in the DOM?

Event bubbling is a DOM event accessing method where the event first triggers on the innermost target element.

Question-5: How do you create, add, and remove elements using JavaScript?
